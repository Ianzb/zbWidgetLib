# [zb组件库（zbWidgetLib）](https://ianzb.github.io/project/zbWidgetLib.html)

zb组件库（zbWidgetLib）是一个PySide6组件库，提供了一些便捷组件，目前用于zb小程序等个人Python项目中。

## 链接

[Github](https://github.com/Ianzb/zbWidgetLib/)
